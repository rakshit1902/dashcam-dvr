# OkDriver DVR History Playback

A DVR-style history playback feature built for the OkDriver Smart Dashcam Platform.

## Features
- Clickable timeline to jump between clips
- Continuous auto-play between clips
- Dual camera view (front + rear) side by side

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173

## Tech Stack
- React
- Vite
